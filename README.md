<h1 align="center">Star Wars Explorer</h1>

A [Coderhouse](https://www.coderhouse.com/) challenge to explore the Star Wars universe. The app consumes the [SWAPI](https://swapi.dev/) (Star Wars API) to display information about characters, planets, and starships from the saga.

### Features

- **Characters Catalog**: Explore the heroes and villains of the galaxy
- **Planets**: Discover the worlds of the Star Wars universe
- **Starships**: Learn about the iconic ships from the saga
- **Holocron Chat**: An AI chatbot that acts as a Jedi Master Archivist, answering questions about the Star Wars universe
- **Pagination**: Smooth navigation through listings
- **Responsive Design**: Interface adapted to all devices

## Getting Started

### Prerequisites

- Node.js 18+
- npm, yarn, pnpm or bun

### Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/sw-coderhouse.git
cd sw-coderhouse
```

2. Install dependencies:

```bash
npm install
```

3. Set up environment variables:

```bash
cp .env.example .env
```

Edit the `.env` file and add your Google AI API key:

```
GOOGLE_GENERATIVE_AI_API_KEY=your_api_key_here
```

4. Start the development server:

```bash
npm run dev
```

5. Open [http://localhost:3000](http://localhost:3000) and travel to a galaxy far, far away.
